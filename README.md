# Демо-интерфейс варианта 2

Это локальный веб-интерфейс на Streamlit для демонстрации работы с графом знаний и документами из Neo4j.

## Что умеет
- Поиск дел по номеру, статье УК и суду
- Просмотр карточки дела
- Просмотр фрагмента текста документа
- Визуализация локального графа вокруг выбранного дела
- Вывод базовых метрик: число узлов/связей, топ статей, топ судов

## Файлы
- `app.py` — основной интерфейс
- `requirements.txt` — зависимости

## Установка
```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
```

## Переменные окружения
PowerShell:
```powershell
$env:NEO4J_URI="bolt://localhost:7687"
$env:NEO4J_USER="neo4j"
$env:NEO4J_PASSWORD="ваш_пароль"
```

## Запуск
```bash
streamlit run app.py
```

## Сценарий показа
1. Открыть вкладку `Поиск`
2. Найти дело по статье или суду
3. Скопировать номер дела
4. Открыть вкладку `Карточка дела`
5. Показать атрибуты документа и локальный граф
6. Открыть вкладку `Метрики`
